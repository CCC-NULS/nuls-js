# QR Code

We use the QRCode npm package to handle creating the QR code for us, more specifically we use the nodejs implementation.

# Create a QR Code URL

ADDRESS can be either an address or a private key. Developers who read the QRCode should be able to read the QR code and if it's an address they can provide options for adding the contact or send funds, if it's a private key then the can transfer the funds, save or open the account.

OPTIONS is any extra URL parameters added onto the address URL QR code. For example { amount: 1 } can be added so the reader is told the amount of NULS to send in the transaction. Read more options on the documentation.

QRCODE_PACKAGE_OPTIONS provide any additional QRCode package options.

import { QRCode } from 'nuls-js';

QRCode.create(ADDRESS, OPTIONS, QRCODE_PACKAGE_OPTIONS).then((url) =>
{
    console.log(url);
})
.catch((err) =>
{
    console.error(err);
});
